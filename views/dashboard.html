API Reseller Methods

Integrate OTP activation endpoints inside your panels and Telegram bots using standard GET requests.

1. Check Reseller Balance
Request Endpoint (GET):
http://localhost:3000/api/v1/info?key=YOUR_API_KEY
2. Get Verification Number
Request Endpoint (GET):
http://localhost:3000/api/v1/get?key=YOUR_API_KEY&id=PRODUCT_ID
Replace PRODUCT_ID with the service code (e.g. 337 for Telegram).
3. Retrieve SMS message code
Request Endpoint (GET):
http://localhost:3000/api/v1/msg?key=YOUR_API_KEY&id=PRODUCT_ID&number=PHONE_NUMBER
